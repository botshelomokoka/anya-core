# Block Types

This document describes the different types of blocks handled by the Anya Bitcoin integration.

## Block Categories

### 1. Main Chain Blocks
- Genesis block
- Regular blocks
- Checkpoint blocks
- Latest blocks

### 2. Fork Blocks
- Temporary fork blocks
- Orphan blocks
- Stale blocks
- Reorganization blocks

### 3. Special Blocks
- Segwit blocks
- Taproot blocks
- Mining blocks
- Block templates

### 4. Block Data
- Block headers
- Block bodies
- Merkle trees
- Witness data
